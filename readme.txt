== Description ==

This is just simple plugin to redirect your search pages' url when got 404 into nice search slug.

== Installation ==

1. Upload 'nice-search-slugzip' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Plugin setting to set your nice search slug.
4. Save your nice search and see the result when your search page got 404 ( pages not found ).
